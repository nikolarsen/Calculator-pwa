# Калькулятор (PWA)

Простой PWA-калькулятор с тёмной темой. Чтобы развернуть:
1. Загрузите файлы в корень репозитория на GitHub.
2. Включите GitHub Pages: Settings → Pages → Branch: main, Folder: /(root) → Save.
3. Откройте URL и добавьте на главный экран (Safari: Поделиться → Добавить на главный экран).

Файлы:
- index.html — интерфейс
- styles.css — стили
- app.js — логика калькулятора (работает офлайн)
- manifest.webmanifest — PWA-манифест
- sw.js — service worker (кеш)
- /icons/ — иконки приложения (замените на нормальные png)
